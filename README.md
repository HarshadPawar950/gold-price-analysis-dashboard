# Gold Price Analysis Dashboard (2016–2025)

## Project Overview

This project analyzes historical gold price data using SQL and Power BI to identify trends, moving averages, and volatility.

## Tools Used

* SQL (Data Cleaning & Analysis)
* Power BI (Dashboard Visualization)
* Excel/CSV (Dataset)
* PYTHON

## Key Insights

* Gold prices show steady growth from 2016–2025
* Moving averages (MA20, MA50, MA200) help identify trends
* Trading volume fluctuates significantly during high volatility periods

## Dashboard Features

* Monthly average gold price
* Yearly gold price trend
* Volume analysis
* Moving average comparison
